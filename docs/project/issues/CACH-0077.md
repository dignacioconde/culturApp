## Alcance

Incluido:

- Shell privado, contenedores de pagina y superficies base.
- Estados visuales de navegacion desktop/mobile ya existentes.
- Botones, chips o controles compartidos solo cuando el cambio sea de token, contraste, borde, foco o espaciado menor.

Fuera de alcance:

- Importar shadcn/Radix, TanStack Start o componentes completos de `artistic-rhythm`.

- Dark mode.
- Supabase, RLS, hooks, datos o formulas financieras.

## Criterios de aceptacion

- [x] El shell privado mantiene las rutas actuales y usa tokens semanticos para fondo, texto, bordes, acento y foco.

- [x] Los targets tactiles de acciones frecuentes se mantienen razonables en mobile.
- [x] `/dashboard` y `/work` no presentan solapes ni saltos visuales obvios en mobile y desktop.
- [x] No se anade ninguna dependencia UI nueva ni se importa stack Lovable.
