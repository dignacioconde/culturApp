# CACH-0090 — Clarificar Agenda y Plan anual

## Objetivo

Hacer que el usuario entienda cuándo usar Agenda y cuándo usar Plan anual sin reintroducir una vista unificada.

## Alcance

- Cabecera o navegación común entre `/calendar/events` y `/calendar/projects`.
- Copy breve: eventos con hora exacta frente a proyectos por rango.
- Leyendas/resúmenes para colores visibles.

## Criterios de aceptacion

- [x] Agenda y Plan anual se pueden cambiar desde la propia pantalla de calendario.
- [x] El texto no promete una sincronización pública que no exista.
- [x] Los colores visibles tienen etiqueta textual o contexto.

## Validacion

- `npm run lint`
- `npm run build`
- Smoke responsive en `/calendar/events` y `/calendar/projects`.

## Resultado

Agenda y Plan anual comparten navegación clara, copy operativo y leyendas/resúmenes para que los colores tengan texto asociado.
