# CACH-0092 — Feed suscribible privado de eventos

## Objetivo

Permitir que cada usuario cree enlaces privados y revocables para suscribir sus eventos de Cachés en calendarios externos compatibles con `.ics/webcal`.

## Alcance

- Tabla `calendar_feeds` con token hasheado.
- RPCs para crear y revocar feeds.
- RPC pública `security definer` para devolver solo campos seguros de eventos por hash de token.
- Edge Function `calendar-feed` que sirve `text/calendar`.

## Criterios de aceptacion

- [x] El token raw no se guarda en base de datos.
- [x] Un feed revocado no devuelve eventos.
- [x] El feed no expone notas, finanzas, datos fiscales, contratantes sensibles ni `user_id`.
- [x] Los eventos usan `UID` estable para evitar duplicados en calendarios externos.

## Plan tecnico

La Edge Function se despliega sin JWT porque los clientes de calendario no mandan sesión de Supabase. La seguridad depende de token privado fuerte, hash SHA-256 en base de datos y revocación. La función calcula el hash del token recibido y llama una RPC acotada que solo devuelve campos permitidos.
