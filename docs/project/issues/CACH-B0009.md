# CACH-B0009 — Inteligencia financiera y features Pro

## Summary

Construir una capa de inteligencia financiera para salud de proyectos, flujo de caja, objetivos, presupuestos, simulaciones y rentabilidad por hora.

## Context

Agrupa #28, #29, #37, #47, #49 y #50.

## Problem

El valor diferencial de Cachés no es solo registrar datos, sino ayudar a decidir si un proyecto compensa, anticipar meses flojos y entender la rentabilidad real.

## Proposed Solution

- Semáforo de salud financiera por proyecto.
- Flujo de caja mensual.
- Objetivos de ingresos mensuales/anuales.
- Presupuestos por categoría de gasto.
- Simulación de escenarios.
- Rentabilidad neta por hora/día al cerrar proyectos.

## Acceptance Criteria

- [ ] La app distingue registro financiero de análisis financiero.
- [ ] Las métricas Pro dependen de datos suficientemente fiables.
- [ ] Las simulaciones no modifican datos reales.
- [ ] La rentabilidad/hora usa liquidación neta cuando exista.
