# CACH-0030 — Homogeneizar diseño con nueva paleta de colores y fuentes

## Summary

Aplicar la paleta de colores y fuentes definida en [[../context/ui-direction-v3-20260504]] a todos los componentes y vistas que faltan.

## Problem

Se implementó la nueva dirección visual UI v3, pero mitad de los componentes y vistas no aplican la paleta de colores y tipografías definidas. Esto genera inconsistencia visual.

## Context

Tokens visuales definidos en `ui-direction-v3-20260504`:

- Paleta base: `--paper`, `--ink`, `--slate`, `--red`, `--amber`, `--green`, `--surface`, etc.
- Tipografías: DM Serif Display (display), Instrument Sans (UI), DM Mono (mono)
- Radios y sombras definidos

## Proposed Solution

1. Auditoria de componentes que faltan por aplicar la paleta
2. Aplicar tokens CSS a cada componente
3. Verificar consistencia visual en desktop y mobile

## Acceptance Criteria

- [ ] Todos los UI components aplican la paleta de colores v3
- [ ] Tipografías consistentes (DM Serif Display, Instrument Sans, DM Mono)
- [ ] Radios y sombras aplicados según tokens
- [ ] Verificación responsive en mobile y desktop
