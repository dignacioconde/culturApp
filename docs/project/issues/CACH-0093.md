# CACH-0093 — UI de sincronización por proveedor

## Objetivo

Dar una forma clara y segura de crear, copiar, abrir y revocar enlaces de sincronización para Apple Calendar, Google Calendar, Outlook y calendarios compatibles.

## Alcance

- Panel en `/calendar/events`.
- Cards por proveedor con copy específico.
- Acciones de crear, copiar enlace, abrir en Apple Calendar y desactivar.
- Aviso de privacidad y de refresco no inmediato.

## Criterios de aceptacion

- [x] El usuario entiende que el enlace es privado y revocable.
- [x] Apple ofrece `webcal://` y fallback de copia.
- [x] Google explica que se añade por URL desde navegador de ordenador.
- [x] Outlook indica que la suscripción puede tardar en refrescar.
- [x] Los enlaces raw solo se muestran al crearse en la sesión actual.

## Validacion

- `npm run lint`
- `npm run build`
- Smoke manual de crear/copiar/desactivar.

## Resultado

Panel de sincronización añadido en `/calendar/events` con cards por proveedor, aviso de privacidad, creación/copia/apertura Apple y desactivación.
