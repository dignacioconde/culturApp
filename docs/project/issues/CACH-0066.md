# CACH-0066 — Checklist compacto de primeros pasos

## Objetivo

Guiar a cuentas vacias o casi vacias hacia las primeras acciones sin crear una tabla nueva de progreso.

## Alcance

- Mostrar checklist compacto en Dashboard cuando hay poca actividad.
- Situarlo antes del bloque "Ahora" para priorizar la guia inicial.
- En movil, mantenerlo colapsado/compacto por defecto para evitar scroll vertical innecesario.
- Inferir progreso desde perfil, proyectos, eventos, ingresos y modo standalone.
- Incluir perfil, primer trabajo, primer cobro, exportacion y uso movil como app.
- No molestar a usuarios con actividad real.

## Criterios de aceptacion

- [x] El checklist aparece en cuentas vacias o casi vacias.
- [x] En Inicio se muestra por encima del bloque "Ahora".
- [x] Los pasos se infieren con hooks existentes.
- [x] Los enlaces llevan a Ajustes, Trabajos, Tus datos u Onboarding.
- [x] En movil muestra una sola accion sugerida y permite desplegar/ocultar el listado completo.
- [x] Usuarios con actividad suficiente no ven el panel.

## Plan tecnico

Crear un componente de onboarding reutilizable y montarlo en Dashboard sin llamadas directas a Supabase desde componentes.

## Validacion

- `npm run lint`
- `npm run test:e2e`
- Revision responsive de Dashboard en movil y desktop.

## Resultado

Implementado y cerrado en `RELEASE-0.1.0-beta.21`. El checklist aparece solo en cuentas vacias o casi vacias, se situa por encima de "Ahora" y en movil muestra una unica accion sugerida con listado desplegable.
