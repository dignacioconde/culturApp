# CACH-0033 — Vista anual en calendario de proyectos

## Objetivo

Cambiar el calendario de proyectos a una vista anual (12 meses) para dar una visión global del año de trabajo cultural.

## Alcance

Incluido:

- Sustituir o añadir una vista anual al calendario actual de proyectos (react-big-calendar u otro).
- La vista anual muestra los proyectos como rangos de color sobre el grid de 12 meses.
- Navegación entre años.

Fuera de alcance:

- Calendarios de eventos (siguen con vista mensual/semana).
- Interacción rápida o filtros avanzados (eso va en CACH-B0007).
- Crear/editar proyectos desde la vista anual.

## Criterios de aceptacion

- [ ] El calendario de proyectos tiene una vista anual que muestra los 12 meses.
- [ ] Los proyectos aparecen como rangos (fecha inicio → fecha fin) sobre el grid anual.
- [ ] Se puede navegar entre años.
- [ ] La vista es responsive (legible en móvil sin scroll horizontal excesivo).
- [ ] `npm run lint` y `npm run build` pasan.

## Validacion

- `npm run lint` y `npm run build`.
- Visual en móvil y desktop.

## Resultado

Released en [[../releases/RELEASE-0.1.0-beta.4|RELEASE-0.1.0-beta.4]]. Se implemento una planificacion anual de proyectos con timeline desktop, Gantt compacto movil, lista mensual y helpers puros testeados para rangos y cruces de anos.
