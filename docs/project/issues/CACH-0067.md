# CACH-0067 — PWA instalable basica y navegacion standalone

## Objetivo

Hacer que Cachés sea instalable como PWA basica y que al abrirse desde el icono mantenga la navegacion interna sin barras del navegador.

## Alcance

- Manifest con `display: standalone`, `scope: "/"`, `start_url`, colores e iconos.
- Meta tags PWA y Apple en `index.html`.
- Service worker para app shell y assets estaticos.
- Smoke postdeploy ampliado para manifest y service worker.

Fuera de alcance: push, notificaciones, recordatorios, resumen semanal y offline de datos con conflictos.

## Criterios de aceptacion

- [x] `/manifest.webmanifest` declara standalone, root scope e iconos 192/512/maskable.
- [x] `sw.js` cachea solo app shell/assets same-origin.
- [x] No se cachean Supabase Auth, REST, Edge Functions ni datos de usuario.
- [x] Se conserva el rewrite SPA de Vercel.
- [ ] Verificacion manual iPhone/Android tras deploy.

## Plan tecnico

Usar assets en `public/`, registrar service worker solo en builds de produccion y mantener la estrategia network-first para navegacion.

## Validacion

- E2E de manifest.
- `npm run build`
- `npm run smoke:postdeploy -- --url=https://app.caches.es` tras deploy.
- Smoke manual iPhone Safari y Android Chrome.

## Resultado

Implementado localmente dentro de beta 21.
