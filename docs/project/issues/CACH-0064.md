# CACH-0064 — Corregir version y copy de consentimiento beta

## Objetivo

Eliminar la referencia obsoleta a beta 8 en consentimiento y tratar `usage_consent_version` como version del texto aceptado, no como numero de release.

## Alcance

- Centralizar `USAGE_CONSENT_VERSION`.
- Sustituir el copy de onboarding y Ajustes por texto de version actual.
- Limpiar `usage_consent_at` y `usage_consent_version` cuando el consentimiento queda desactivado.
- No anadir schema, analitica, PostHog ni tracking.

- [x] No queda `beta-8` ni "beta 8" en el runtime de la app.

- [x] Los cambios de perfil siguen pasando por `useProfile`.
