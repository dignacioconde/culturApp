# CACH-0087 — Unificar admin gates y layout global

## Objetivo

Unificar visualmente gates de perfil/admin, admin de invitaciones y layout global sin cambiar permisos ni flujos.

## Alcance

Incluido:

- Loaders/gates de `App.jsx`, `/admin/invitaciones`, top/bottom bars y avisos layout.
- Estados admin, codigo largo, notices y acciones sensibles visualmente distinguibles.
- Review de seguridad para no alterar permisos ni exposicion de codigos.

Fuera de alcance:

- Cambiar auth, roles, Supabase, funciones de invitacion o mutaciones admin.
- Anadir funcionalidades Lovable nuevas.

## Criterios de aceptacion

- [x] Gates de perfil/admin usan el sistema visual Lovable y `Button`.
- [x] `/admin/invitaciones` conserva permisos, flujos y copy de seguridad.
- [x] Bottom nav/action bars y toasts no se pisan en rutas secundarias.

## Resultado

Implementada por worker frontend en `App.jsx`, admin de invitaciones y layout global: gates, loaders, top/bottom bars, sidebar, estados admin y codigo largo usan tokens Lovable sin cambiar auth, roles ni mutaciones.
