# Gotcha — 409 en creación de proyecto/evento = fila de pro...

Gotcha — 409 en creación de proyecto/evento = fila de profile faltante

Síntoma: creación de proyecto o evento devuelve error 409.
Causa: el usuario autenticado no tiene fila en public.profiles. El trigger handle_new_user debería crearla al registrarse, pero puede fallar silenciosamente.
Reparación: insertar en public.profiles los IDs que existen en auth.users pero no en profiles, después de verificar que handle_new_user inserta correctamente.

Contexto: esto bloqueó a usuarios reales durante el MVP.
