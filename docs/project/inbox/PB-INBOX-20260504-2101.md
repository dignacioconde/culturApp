# UI compacta: cards/cajas en toda la app

**Problema**: Las tarjetas (cards) y cajas de la UI ocupan demasiado espacio, especialmente en móvil.

**Alcance**:
- EventList / ProjectList
- KPI cards del Dashboard
- Cualquier tarjeta/caja en modales o secciones

**Observación principal**: El Dashboard en móvil debería ser tan compacto como Events y Projects.

**A revisar**:
- Padding interno (actualmente 16px, podría ser 12px)
- Gap entre items (actualmente 12px, podría ser 8px)
- Layout de grid/cols en mobile
- Legibilidad vs. densidad
- Targets táctiles (mínimo 40-44px, según AGENTS.md)
