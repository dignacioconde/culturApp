# Contexto — Stack técnico de Cachés y motivación de cada e...

Contexto — Stack técnico de Cachés y motivación de cada elección

React 19 + Vite 8: ecosistema maduro, build rápido sin configuración compleja.
Tailwind CSS v4: utility-first, prototipar rápido.
Lucide React: iconos limpios y consistentes, 1500+.
React Big Calendar: vista mes/semana/día con drag & drop.
Day.js: fechas ligeras con API limpia.
React Router DOM 7: navegación entre vistas.
Supabase JS: PostgreSQL + Auth + API REST sin servidor propio — elimina la necesidad de backend dedicado para el MVP.
Vercel: deploy automático desde GitHub.

Estas elecciones fueron deliberadas para minimizar infraestructura y maximizar velocidad de iteración en el MVP.
