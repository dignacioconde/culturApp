# ADR — pb:push con --delete sincroniza borrados al vault d...

ADR — pb:push con --delete sincroniza borrados al vault de iCloud

Decisión: el comando pb:push tiene un flag --delete que borra automáticamente del vault de Obsidian/iCloud los archivos que ya no existen en el repo.

Motivación: antes (v1) el sync era unidireccional para escrituras — nunca borraba. Esto generó duplicados huérfanos en el vault (CACH-026/CACH-0026, CACH-B001..B014/CACH-B0001..B0014) que hubo que limpiar manualmente.

Uso correcto:
- npm run pb:push → solo exporta archivos nuevos/modificados (seguro, no destructivo)
- npm run pb:push -- --delete → exporta Y borra del vault los que ya no están en el repo

Implementado en PR #49 (issue #48). Script: scripts/product-brain-sync.mjs.
