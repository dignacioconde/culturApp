# Templates

Plantillas para crear entradas consistentes en Obsidian.

## Canonicas

- [[ISSUE_TEMPLATE]] — Issue Markdown `CACH-*`.
- [[./RELEASE_TEMPLATE|RELEASE_TEMPLATE]] — Release con rama, checklist y release notes.
- [[ADR_TEMPLATE]] — Decision importante de producto, arquitectura, datos, UX o workflow.

## Compatibilidad

Los archivos legacy en minusculas redirigen a estas plantillas canonicas para evitar formatos paralelos.
